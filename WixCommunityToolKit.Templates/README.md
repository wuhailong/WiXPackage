## 安装项目模板
仅需一次,段名称Wix.Kit
```
dotnet new --install Wix.CommunityToolKit
```

![](\Images\06224.PNG)

## 卸载项目

```
dotnet new -u --uninstall Wix.CommunityToolKit
```

![](\Images\06224_2.PNG)

##  创建项目

```
dotnet new Wix.Kit
```
